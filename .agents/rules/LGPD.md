**POLÍTICA DE PRIVACIDADE**

Esta Política de Privacidade descreve como as informações são coletadas, utilizadas e protegidas ao utilizar o aplicativo.

**1. Coleta de Informações**

O aplicativo pode coletar informações fornecidas diretamente pelo usuário, incluindo:

- nome
- telefone
- registros de clientes
- valores de dívidas ou vendas a prazo
- histórico de pagamentos
- dados de login e conta

Essas informações são inseridas diretamente pelo usuário com a finalidade de organizar e gerenciar registros financeiros.

**2. Finalidade do Tratamento de Dados**

Os dados coletados são utilizados exclusivamente para:

- permitir o funcionamento do aplicativo
- armazenar registros de controle financeiro
- fornecer relatórios e histórico de transações registradas
- melhorar a experiência de uso da plataforma

**3. Responsabilidade sobre os Dados Inseridos**

O usuário é responsável pelos dados pessoais que inserir na plataforma, incluindo informações relacionadas a seus clientes ou terceiros.

O usuário declara que possui base legal para registrar essas informações conforme a **Lei Geral de Proteção de Dados (Lei nº 13.709/2018)**.

**4. Compartilhamento de Dados**

A plataforma não vende, aluga ou compartilha dados pessoais com terceiros, exceto quando:

- necessário para funcionamento técnico do serviço
- exigido por obrigação legal ou ordem judicial

**5. Segurança das Informações**

São adotadas medidas técnicas e administrativas razoáveis para proteger as informações armazenadas contra acesso não autorizado, perda ou alteração.

**6. Armazenamento de Dados**

Os dados permanecerão armazenados enquanto a conta do usuário estiver ativa ou pelo período necessário para cumprimento de obrigações legais.

**7. Direitos do Usuário**

Nos termos da **Lei Geral de Proteção de Dados (Lei nº 13.709/2018)**, o usuário pode solicitar:

- acesso aos dados armazenados
- correção de informações
- exclusão de dados quando aplicável

**8. Alterações nesta Política**

Esta Política de Privacidade poderá ser atualizada periodicamente para refletir melhorias ou mudanças no serviço.
